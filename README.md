# WeatherCheck - Fullstack Weather App

WeatherCheck – Full-stack weather app using Node.js &amp; Express.  My first node based web-app.
A simple full-stack weather application built to understand
frontend–backend communication, internal APIs, and external API integration.

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- External API: OpenWeather API

## Features

- Fetches real-time weather data
- Backend API acts as a proxy
- Environment variables for Security
- Loading spinner during API calls

## Learning Outcomes

- Frontend & Backend architecture
- Internal vs External APIs
- Async JavaScript
- CORS handling
